###/hypem

This'll pull track data from [The Hype Machine](http://hypem.com)

Valid usage:

    /hypem genre <genre name>
    /hypem search <search term>
    /hypem artist <artist name>
    /hypem popular
    /hypem new

Examples:

    /hypem genre dubstep
    /hypem search party rock
    /hypem artist C2C
    /hypem popular
    /hypem new
